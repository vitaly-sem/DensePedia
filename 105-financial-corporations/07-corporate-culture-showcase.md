# Корпоративная культура в больших корпорациях — сравнение и комментарии

> Как устроена культура в крупнейших корпорациях мира: технологии, финансы, промышленность.

---

## 1. Технологические гиганты

### Google (Alphabet)

| Аспект | Описание |
|---|---|
| **Тип (Cameron-Quinn)** | Адхократия + Клан |
| **Ценности** | «Don't be evil», десятичная мысль (10x), data-driven |
| **Плюсы** | Отличные условия (еда, спортзал), психологическая безопасность, 20% времени |
| **Минусы** | Бюрократия с ростом, политика, «dead sea effect» |
| **Известно** | Project Aristotle (эффективность команд), OKR, culture of psychological safety |

**Комментарий**: Google создал культуру, где ошибки — часть процесса, а данные — основа решений. Но с ростом до 150K+ сотрудников культура стала размываться.

### Netflix

| Аспект | Описание |
|---|---|
| **Тип** | Рынок + Адхократия |
| **Ценности** | Freedom & Responsibility, Context not Control, Radical Candor |
| **Плюсы** | Высочайшая эффективность, лучшие люди, высокая зарплата |
| **Минусы** | Жёсткий отбор («adequate performance = severance»), выгорание |
| **Известно** | Netflix Culture Deck (124 слайда, 2009), «keeper test» |

**Комментарий**: Netflix построил культуру «взрослых людей» — максимум свободы и максимум ответственности. Но не каждому такое подходит. Увольняют без сожаления — «достаточная работа — это не работа».

### Amazon

| Аспект | Описание |
|---|---|
| **Тип** | Рынок |
| **Ценности** | Customer obsession, frugality, bias for action, high standards |
| **Плюсы** | Дисциплина, результативность, долгосрочное мышление |
| **Минусы** | Печально известный «stack ranking», переработки, текучка |
| **Известно** | 14 Leadership Principles, «Day 1» менталитет, six-pagers |

**Комментарий**: Amazon — культура интенсивной конкуренции. Работают люди, готовые к жёсткой обратной связи и постоянному давлению. Critics note: high burnout rate.

### Microsoft (эпоха Satya Nadella)

| Аспект | Описание |
|---|---|
| **Тип** | Клан → Адхок (трансформация) |
| **Ценности** | Growth mindset, diversity, empathy, innovation |
| **Плюсы** | Культура обучения, открытость к open source |
| **Минусы** | Бюрократия, legacy-культура |
| **Известно** | Трансформация из «know-it-all» в «learn-it-all», отказ от stack ranking |

**Комментарий**: Самая впечатляющая трансформация культуры в тех-индустрии. При Балмере — политика и конкуренция, при Наделла — эмпатия и рост. Результат: капитализация с $300B до $3T.

### Apple

| Аспект | Описание |
|---|---|
| **Тип** | Рынок + Адхок |
| **Ценности** | Perfectionism, secrecy, simplicity |
| **Плюсы** | Продукты высочайшего качества |
| **Минусы** | Секретность, культура страха, политика |
| **Известно** | «DRI» (Directly Responsible One), культура секретности |

**Комментарий**: Культура Apple построена вокруг продукта. Секретность — легендарна (сотрудники не знают, над чем работают соседи). Минус: после Джобса — бюрократизация.

## 2. Финансовые корпорации

### Goldman Sachs

| Аспект | Описание |
|---|---|
| **Тип** | Рынок + Иерархия |
| **Ценности** | Client first, partnership, long-term greedy |
| **Плюсы** | Престиж, доход, лучшие клиенты |
| **Минусы** | «Культура волка», 100-часовая неделя, политика |
| **Известно** | Partnership model (до 1999), 14 принципов, «revolving door» |

**Комментарий**: Goldman — элитарная культура. Работа там = знак качества на всю жизнь. Но культура «любой ценой» привела к скандалам (кризис 2008, 1MDB).

### JPMorgan Chase

| Аспект | Описание |
|---|---|
| **Тип** | Иерархия + Рынок |
| **Ценности** | Спокойствие, надёжность, долгосрочность |
| **Плюсы** | Стабильность, Jamie Dimon (культурный лидер) |
| **Минусы** | Консерватизм, громоздкость |
| **Известно** | Dimon — «the last man standing» 2008 |

**Комментарий**: Самая сильная культура среди банков. Dimon лично участвует в обучении новых сотрудников. Культура risk management ≠ risk avoidance.

### BlackRock

| Аспект | Описание |
|---|---|
| **Тип** | Рынок + Адхок |
| **Ценности** | One BlackRock, technology-first, risk management |
| **Плюсы** | Aladdin (технологическая платформа), глобальное влияние |
| **Минусы** | Политика, концентрация власти |
| **Известно** | Larry Fink — «purpose before profit», Aladdin |

**Комментарий**: BlackRock — не просто финансовая компания, это тех-компания с финансами. Культура data-driven, инженерная. Влияние BlackRock на глобальную экономику огромно.

## 3. Промышленные гиганты

### Toyota

| Аспект | Описание |
|---|---|
| **Тип** | Иерархия + Клан |
| **Ценности** | Kaizen (постоянное улучшение), respect for people, Genchi Genbutsu |
| **Плюсы** | Качество, эффективность, уважение к сотрудникам |
| **Минусы** | Медленные изменения |
| **Известно** | Toyota Production System, Lean, Andon cord, Kaizen |

**Комментарий**: Культура Toyota — эталон для производственных компаний. Kaizen не про оптимизацию, а про уважение к людям (они — источник улучшений). Andon — любой может остановить конвейер.

### Berkshire Hathaway

| Аспект | Описание |
|---|---|
| **Тип** | Клан |
| **Ценности** | Honesty, long-term thinking, decentralized |
| **Плюсы** | Минимальная бюрократия, доверие к менеджерам |
| **Минусы** | Зависимость от Баффета (97 лет) |
| **Известно** | Annual meeting — «Woodstock for capitalists» |

**Комментарий**: Уникальная культура — 90+ дочерних компаний с полной автономией. Баффет не даёт указаний, он «подбирает» менеджеров и доверяет им. Результат: $900B капитализации.
