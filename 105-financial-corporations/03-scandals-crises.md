## 1. Финансовые кризисы

### Великая депрессия (1929–1939)

| Параметр | Значение |
|---|---|
| **Причина** | Спекулятивный пузырь, маржинальная торговля, банковская паника |
| **Падение рынка** | Dow Jones: -90% (1929–1932) |
| **Банкротства** | >9000 банков |
| **Последствия** | Glass-Steagall Act (1933), FDIC, SEC |

### Кризис 2008 (Мировой финансовый кризис)

| Параметр | Значение |
|---|---|
| **Причина** | Субстандартная ипотека (subprime), CDO, credit default swaps |
| **Триггер** | Lehman Brothers (15.09.2008) — банкротство $600 млрд |
| **Спасение** | TARP $700 млрд (правительство США) |
| **Стоимость** | $5-15 трлн потерь (мировая экономика) |

**Ключевые события**:

| Дата | Событие | Значение |
|---|---|---|
| 2007 | Bear Stearns — collapse | Первый признак кризиса |
| 07.09.2008 | Fannie Mae, Freddie Mac — национализация | $6 трлн ипотек |
| 15.09.2008 | Lehman Brothers — банкротство | Не спасали — кризис доверия |
| 16.09.2008 | AIG — спасение $85 млрд | «Too big to fail» |
| 03.10.2008 | TARP — закон | $700 млрд на спасение |

**Кто пострадал**: миллионы потеряли дома, $17 трлн богатства домохозяйств уничтожено

**Кто не пострадал**: банкиры получили бонусы (AIG — $165 млн после спасения)

## 2. Крупнейшие скандалы

### Enron (2001)

| Параметр | Значение |
|---|---|
| **Сфера** | Энергетика |
| **Мошенничество** | Забалансовые SPE (Special Purpose Entities) для сокрытия долгов |
| **Убыток инвесторов** | $74 млрд |
| **Последствия** | Разрушение Arthur Andersen (аудитор), закон Sarbanes-Oxley |

**Схема**: Enron создавал «партнёрств» (LJM, Raptors), которым продавал активы по завышенной цене, записывая прибыль. Компании были гарантированы Enron → фактически кредит.

> **Ключевая цитата**: «Enron управляла прибылью, а не бизнесом» (Sherron Watkins, whistleblower)

### Bernie Madoff (2008)

| Параметр | Значение |
|---|---|
| **Схема** | Пирамида Понци (крупнейшая в истории) |
| **Убыток** | $65 млрд (номинально), $18 млрд (реальный убыток) |
| **Длительность** | ~1990–2008 (18+ лет) |
| **Количество жертв** | Тысячи (включая благотворительные фонды, пенсионеров) |
| **Наказание** | 150 лет тюрьмы (умер 2021) |

**Почему не поймали раньше**:
- Madoff был председателем NASDAQ
- Его компания была «верняком» — стабильные 10-12% в год
- SEC получала предупреждения (Гарри Маркопулос, 2000, 2005) — игнорировали 5 раз

### LIBOR-скандал (2012)

| Параметр | Значение |
|---|---|
| **Что** | Манипуляция межбанковской ставкой LIBOR |
| **Участники** | Barclays, UBS, RBS, Deutsche Bank, Citi |
| **Штрафы** | >$9 млрд суммарно |
| **Последствия** | LIBOR отменён (2023), заменён на SOFR |

**Схема**: Трейдеры просили коллег занижать/завышать LIBOR, чтобы получить прибыль по деривативам. Barclays: 2005–2009 систематическая манипуляция.

### Panama Papers (2016)

| Параметр | Значение |
|---|---|
| **Источник** | Утечка 11.5 млн документов Mossack Fonseca (панамская юридическая фирма) |
| **Последствия** | Отставка премьера Исландии (Сигмюндюр Гюннлёйгссон) |
| **Результаты** | 150+ расследований, возврат $1.2 млрд налогов |

> **Факт**: 214 000 офшорных компаний, клиенты — 12 действующих и бывших глав государств, 128 политиков

### GameStop / WallStreetBets (2021)

| Параметр | Значение |
|---|---|
| **Что** | Розничные инвесторы (Reddit) против шортистов (Melvin Capital, Citron) |
| **Механизм** | Short squeeze: GameStop с $4 до $483 |
| **Убыток шортистов** | ~$20 млрд (Melvin Capital закрылся) |
| **Значение** | Эрозия доверия к «справедливым рынкам» |
