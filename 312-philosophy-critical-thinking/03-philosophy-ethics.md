## Эпистемология — теория познания

### Что такое знание?

Классическое определение (Plato): **Justified True Belief**.

```
S знает, что P, если:
1. P истинно
2. S верит, что P
3. S имеет обоснование для веры в P
```

### Gettier-проблема

Пример: Вы смотрите на часы, которые показывают 14:00. Часы сломаны, но сейчас действительно 14:00.

- Вы верите, что 14:00 ✅
- Это правда ✅
- У вас есть основание ✅
- Но это не знание, а случайность

### Источники знания

| Источник | Надёжность | Пример |
|----------|------------|--------|
| **Perception** (чувства) | Низкая (иллюзии) | «Палка в воде кажется сломанной» |
| **Reason** (разум) | Высокая | Математика, логика |
| **Testimony** (свидетельство) | Средняя | «Мне сказали...» |
| **Intuition** | Низкая | «Чувствую, что...» |

## Метафизика

### Реализм vs Идеализм

| Позиция | Суть |
|---------|------|
| **Наивный реализм** | Мир таков, каким мы его видим |
| **Критический реализм** (Kant) | Мир существует, но мы познаём его через призму чувств |
| **Идеализм** (Berkeley) | Реальность — конструкт сознания |
| **Солипсизм** | Существует только моё сознание |

### Free Will vs Determinism

```
Determinism: Всё предопределено (законы физики)
Compatibilism: Свобода воли совместима с детерминизмом
Libertarianism: Свободная воля реальна (indeterminism)
```

**Практический вывод:** Даже если детерминизм истинен, мы **вынуждены** принимать решения так, как будто свобода воли существует.

## Этика

### Основные теории

| Теория | Принцип | Пример |
|--------|---------|--------|
| **Утилитаризм** (Bentham, Mill) | Максимум счастья для максимума людей | «Приносим в жертву один сервис для спасения десяти» |
| **Деонтология** (Kant) | Есть моральные правила, независимо от последствий | «Мы не врём пользователям, даже если это выгодно» |
| **Этика добродетели** (Aristotle) | «Каким человеком мне быть?» | «Будь честным, смелым, справедливым» |
| **Контрактуализм** (Rawls) | Мораль — это то, на что согласились бы рациональные люди | Социальный контракт |

### Этические дилеммы в IT

| Дилемма | Вопрос |
|---------|--------|
| **Privacy vs Feature** | «Мы можем собирать больше данных для лучшего UX, но это нарушает приватность» |
| **Speed vs Quality** | «Выкатить багфикс сейчас (может сломать другие фичи) или ждать QA (пользователи страдают)?» |
| **AI Bias** | «Наша ML-модель отсеивает 80% женщин → прибыль выше» |
| **Dark Patterns** | «Кнопка "Отменить подписку" спрятана — меньше отписок» |
| **Whistleblowing** | «Я вижу нарушение закона в коде, но начальство молчит» |

### Этический чек-лист для инженера

```
1. Нарушает ли это закон?
2. Нарушает ли это права людей?
3. Буду ли я готов объяснить это решение публично?
4. Хотел бы я, чтобы это решение применяли ко мне?
5. Есть ли альтернатива без этического конфликта?
```

## Философия сознания (Philosophy of Mind)

### Mind-Body Problem

| Позиция | Суть |
|---------|------|
| **Дуализм** (Descartes) | Сознание и мозг — разные субстанции |
| **Физикализм** | Сознание = работа мозга |
| **Функционализм** | Сознание — это функциональные состояния (может быть на разных носителях) |
| **Панпсихизм** | Сознание присуще всей материи |

### Hard Problem of Consciousness (Chalmers)

```
Easy problems: Как мозг обрабатывает информацию? (решаемо)
Hard problem: Почему обработка информации сопровождается субъективным опытом (qualia)?
```

### Отношение к AI

- **Strong AI** (Searle): компьютер может обладать сознанием
- **Chinese Room**: синтаксис ≠ семантика (программа не понимает)
- **LLM понимают?** — открытый философский вопрос
