# Христианство — история и изменение традиций

> Эпоха: от I в. н.э. до современности

---

## Хронология

| Период | Событие | Изменение |
|---|---|---|
| **~4 до н.э. — 30 н.э.** | Жизнь Иисуса | Проповедь, распятие (ок. 30 н.э.), воскресение |
| **~30–70** | Апостольский век | Пётр, Павел, миссия к язычникам. Иерусалимский собор (50 н.э.) |
| **~70–313** | Ранняя церковь, гонения | Канон Нового Завета (IV в.), апологеты, мартиролог |
| **313** | Миланский эдикт (Константин) | Легализация христианства |
| **325** | Первый Никейский собор | Символ веры, осуждение арианства |
| **380** | Фессалоникийский эдикт (Феодосий) | Христианство = государственная религия Рима |
| **~400–500** | Августин | «Град Божий», доктрина первородного греха |
| **451** | Халкидонский собор | Божественная и человеческая природа Христа |
| **~500–1054** | Разделение церквей (Великая схизма) | Рим (католики) vs Константинополь (православные) |
| **1054** | Формальная схизма | Взаимное отлучение |
| **1096–1271** | Крестовые походы | Попытка отвоевать Святую землю |
| **~1200–1300** | Схоластика (Фома Аквинский) | Рациональное богословие, Summa Theologica |
| **1517** | 95 тезисов Лютера | Начало Реформации |
| **~1520–1560** | Протестантизм | Лютеранство, кальвинизм, англиканство |
| **1545–1563** | Тридентский собор | Контрреформация, католические догматы |
| **~1870** | I Ватиканский собор | Папская непогрешимость |
| **~1962–1965** | II Ватиканский собор | Месса на национальных языках, экуменизм |
| **~2000–н.в.** | Современное христианство | Рост евангелизма, секуляризация в Европе |

## Основные ветви христианства

### Католицизм (~1.3 млрд)

| Особенность | Детали |
|---|---|
| **Глава** | Папа Римский (епископ Рима) |
| **Догматы** | Непогрешимость папы (1870), непорочное зачатие Марии (1854), чистилище |
| **Таинства** | 7 таинств: крещение, причащение, конфирмация, исповедь, брак, священство, елеосвящение |
| **Расколы** | Восточные католики (униаты) — признают папу, но сохраняют обряды |

### Православие (~260 млн)

| Особенность | Детали |
|---|---|
| **Глава** | Нет единого главы. Поместные церкви (Московская, Константинопольская и др.) |
| **Синод** | Соборное управление |
| **Отличия от католицизма** | Filioque (исходит только от Отца), опресноки, без чистилища |
| **Иконопочитание** | VII Вселенский собор (787) — восстановлено |
| **Старообрядчество** (1666) | Раскол из-за реформы Никона (троеперстие, богослужебные книги) |

### Протестантизм (~900 млн)

| Течение | Возникло | Особенность |
|---|---|---|
| **Лютеране** | 1517 | Только Писание (sola scriptura), только верой (sola fide) |
| **Кальвинисты** | 1536 | Предопределение, Женева |
| **Англикане** | 1534 | Король — глава церкви, компромисс католицизма и протестантизма |
| **Баптисты** | 1609 | Крещение по вере (не младенцев) |
| **Методисты** | 1738 | Джон Уэсли, личное обращение |
| **Пятидесятники** | 1906 | Дары Святого Духа, говорение на языках |
| **Евангелисты** | XX в. | Современное евангелическое движение |

## Ключевые изменения традиций

| Изменение | Когда | Что произошло |
|---|---|---|
| **Отказ от субботы** | II в. | Воскресенье вместо субботы как день отдыха |
| **Целибат священников** | IV в. (постепенно) | В католицизме — обязательный, в православии — для епископов |
| **Иконопочитание** | 787 | VII собор: иконы можно почитать, но не поклоняться |
| **Filioque** | 589 (Толедо) → 1054 | Спор об исхождении Святого Духа (от Отца и Сына) |
| **Чистилище** | XII–XIII вв. | Католическая доктрина (отвергается православием и протестантами) |
| **Индульгенции** | Средневековье | Торговля → Реформация Лютера |
| **Месса на латыни** | 1962–1965 | II Ватикан: переход на национальные языки |
| **Женское священство** | XX в. | Протестанты → да, католики/православные → нет |
| **Однополые браки** | XXI в. | Либеральные протестанты → благословение, католики/православные → нет |

## Факты

- **Христианство — крупнейшая религия мира**: ~2.4 млрд (33% населения)
- **Самая быстрорастущая ветвь**: пятидесятники (с 0 в 1900 до ~600 млн)
- **Самый древний Новый Завет**: Синайский кодекс (IV в.), найден в монастыре Св. Екатерины
- **Библия — самая продаваемая книга**: ~5 млрд копий
- **28 000 протестантских деноминаций** (по данным Center for the Study of Global Christianity)
