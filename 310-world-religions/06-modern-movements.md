# Новые религиозные движения

> Эпоха: от XV в. до современности

---

## 1. Сикхизм (XV–XVI вв.)

| Параметр | Значение |
|---|---|
| **Основатель** | Гуру Нанак (1469–1539) |
| **Регион** | Пенджаб (северная Индия) |
| **Последователей** | ~25 млн |
| **Священный текст** | Гуру Грантх Сахиб |
| **Идея** | Единый Бог, равенство всех людей, отрицание каст |

### Изменения

| Событие | Когда | Изменение |
|---|---|---|
| 10 гуру | 1469–1708 | Устная → письменная традиция |
| Хальса (братство) | 1699 (Гуру Гобинд Сингх) | Воинская община, 5 К (кэш, кханга, кара, качхера, кирпан) |
| Гуру Грантх Сахиб | 1708 | Книга заменила живого гуру |

## 2. Бахаи (XIX в.)

| Параметр | Значение |
|---|---|
| **Основатель** | Бахаулла (1817–1892) |
| **Регион** | Персия (Иран) |
| **Последователей** | ~5-8 млн |
| **Идея** | Единство Бога, единство религий, единство человечества |

### Изменения традиции

| От | К | Изменение |
|---|---|---|
| Шиитский ислам | Новая религия | Баб (1844) → Бахаулла (1863) |
| Национальная | Глобальная | Центр в Хайфе, всемирное административное здание |

## 3. Мормоны (Церковь Иисуса Христа Святых последних дней, 1830)

| Параметр | Значение |
|---|---|
| **Основатель** | Джозеф Смит (1805–1844) |
| **Регион** | США (Юта) |
| **Последователей** | ~17 млн |
| **Священный текст** | Книга Мормона + Библия |

### Изменения традиции

| Что изменилось | Когда | Суть |
|---|---|---|
| Многожёнство (полигамия) | 1843–1890 | Отменено для приёма Юты в США |
| Практика крещения за мёртвых | Продолжается | Генеалогия — важнейшая практика |
| Отношение к чёрным | 1978 | До 1978 — запрет на священство для чернокожих |

## 4. Свидетели Иеговы (1870-е)

| Параметр | Значение |
|---|---|
| **Основатель** | Чарльз Рассел |
| **Регион** | США (Бруклин) |
| **Последователей** | ~8.7 млн |
| **Идея** | Только Библия, отрицание Троицы |

### Изменения традиции

| Что изменилось | Когда | Суть |
|---|---|---|
| Отказ от переливания крови | 1945 | Доктрина, основанная на Деяния 15:29 |
| Ожидание конца света | 1914, 1925, 1975 | Предсказывали, переносили |
| Отказ от политики/военной службы | XX в. | Гонения во многих странах |

## 5. Нью-эйдж (XX в.)

| Параметр | Значение |
|---|---|
| **Период** | 1960-е – н.в. |
| **Центры** | США, Европа |
| **Идея** | Синтез восточной философии, эзотерики, психологии, экологии |

### Элементы

- Реинкарнация (из индуизма/буддизма)
- Кристаллы, астрология, таро
- Экосознание, «Новая эра» (Эра Водолея)
- Медитация, чакры, аура
- Популярность: ~1-2% в западных странах

## Факты о современных религиозных изменениях

| Тенденция | Описание |
|---|---|
| **Секуляризация** | Европа: падение религиозности (Эстония — 16% верят в Бога) |
| **Рост евангелизма** | Африка, Латинская Америка: взрывной рост |
| **Деизм / «духовность без религии»** | «Я spiritual, но не religious» — ~30% в США |
| **Цифровые религии** | Онлайн-службы, приложения для молитв, ИИ-проповедники |
| **Атеизм / агностицизм** | ~16% населения мира, растёт |
