# Ислам — история и изменение традиций

> Эпоха: от VII в. н.э. до современности

---

## Хронология

| Период | Событие | Изменение |
|---|---|---|
| **570–610** | Жизнь Мухаммеда до пророчества | Купец в Мекке |
| **610** | Первое откровение (Ночь Могущества) | Через архангела Джабраила (Гавриила) |
| **610–622** | Проповедь в Мекке | Гонения, мало последователей |
| **622** | **Хиджра** (переселение в Медину) | Начало исламского календаря, первое исламское государство |
| **622–632** | Медина: лидер, войны | Коран, законы, конфликт с Меккой |
| **630** | Завоевание Мекки | Кааба — центр ислама |
| **632** | Смерть Мухаммеда | Кризис преемственности |
| **632–661** | Праведные халифы (Абу-Бакр, Умар, Усман, Али) | Распространение ислама, запись Корана |
| **656–661** | Первая фитна (смута) | Убийство Усмана → раскол на суннитов и шиитов |
| **661–750** | Омейяды | Столица Дамаск, арабизация, завоевания до Испании |
| **750–1258** | Аббасиды | Золотой век ислама: наука, философия (Багдад) |
| **~900–1000** | Канонизация хадисов | Бухари, Муслим |
| **~1050–1200** | Суфизм | Мистическое течение, ордены (тарикаты) |
| **1258** | Разрушение Багдада (монголы) | Конец Аббасидского халифата |
| **~1300–1924** | Османская империя | Столица Стамбул, халифат |
| **~1700–1920** | Упадок, колониализм | Раздел Османской империи, ислам в кризисе |
| **1924** | Упразднение халифата | Ататюрк, Турция — светское государство |
| **~1928–н.в.** | Исламизм | Братья-мусульмане, Хомейни, Саудовская Аравия |
| **1979** | Исламская революция в Иране | Шиитское теократическое государство |

## Ключевые изменения традиций

| Изменение | Когда | Что произошло |
|---|---|---|
| **Запись Корана** | 650–656 (при Усмане) | Устный Коран → единый письменный текст |
| **Хадисы** | IX–X вв. | Сборник преданий о Мухаммеде — второй источник законов |
| **Шариат** | IX–X вв. | Формализация исламского права |
| **Иджтихад** | Ок. X в. | «Врата иджтихада закрыты» — прекращение самостоятельного толкования (спорно) |
| **Халифат → султанат** | X в. | Светская власть (султан) отделяется от религиозной (халиф) |
| **Ваххабизм** | XVIII в. | Возврат к «чистому исламу»: запрет мавзолеев, святых, реформы |
| **Исламский модернизм** | XIX–XX вв. | Попытка совместить ислам с наукой и демократией |
| **Женские права** | XX–XXI вв. | Некоторые страны: запрет никаба (Франция), разрешение женщинам быть имамами (Амина Вадуд, 2005) |

## Факты

- **Коран** — 114 сур, ~6236 аятов. Заучивается наизусть миллионами (хафизы)
- **Хадис Бухари** — ~2600 подлинных хадисов из ~600 000 собранных
- **Пять столпов ислама**: шахада (вера), салят (молитва 5 раз), закят (милостыня), саум (пост Рамадан), хадж (паломничество)
- **Джихад**: в Коране — «усилие на пути Аллаха». Великий джихад (духовный) и малый (военный)
- **Ислам — вторая по численности религия**: ~1.9 млрд (24%)
- **Самая быстрая по росту религия** (демографически)
- **В исламе нет официального священства** — любой может быть имамом
