# Борьба с зависимостями

> Механизмы зависимостей, научные методы борьбы.

---

## 1. Что такое зависимость?

**Зависимость (addiction)** — хроническое расстройство мозга, при котором человек продолжает искать вещество/поведение, несмотря на негативные последствия.

### Механизм дофаминового контура

```
Стимул → Дофамин (выброс) → Желание → Поведение → Удовольствие (меньше) → Ещё больше желания
```

- **Толерантность**: со временем нужно больше вещества/поведения для того же эффекта
- **Абстиненция**: без вещества — дискомфорт, тревога, депрессия
- **Триггеры**: контекст, люди, эмоции, стресс

### Виды зависимостей

| Тип | Примеры | Распространённость |
|---|---|---|
| **Химические** | Алкоголь, никотин, опиоиды, стимуляторы, каннабис | ~15-20% пробовали |
| **Поведенческие** | Игры, соцсети, азарт, порно, шопоголизм | ~5-10% |
| **Еда** | Сахар, переедание, компульсивное | ~2-5% |
| **Технологии** | Смартфон, TikTok, YouTube | ~5-15% |

## 2. Принципы работы с зависимостями

### Мотивационное интервьюирование (Miller, Rollnick)

```
Предсозерцание → Созерцание → Подготовка → Действие → Поддержание
```

- Не бороться с отрицанием
- Усиливать внутреннюю мотивацию
- Не навязывать решения

### КПТ при зависимостях

1. **Триггер** (ситуация, эмоция, место)
2. **Мысль** («одна сигарета не повредит»)
3. **Желание** (тяга, craving)
4. **Поведение** (срыв)
5. **Последствие** (вина, стыд → усиление цикла)

**Техника ABC**:
- A — Antecedent (триггер)
- B — Belief (мысль)
- C — Consequence (эмоция/действие)

### Система вознаграждений

- **Заменить** привычку, а не просто убрать
- **Добавить** новый позитивный ритуал на место старого
- **Правило 5 минут**: при тяге подождать 5 минут (срабатывает в 50% случаев)

## 3. Конкретные методы для разных зависимостей

### Алкоголь

| Метод | Доказанность |
|---|---|
| **Анонимные Алкоголики (12 шагов)** | 🟡 Умеренная (сложно изучать) |
| **Медикаменты (налтрексон, дисульфирам, акампросат)** | 🟢 Высокая |
| **КПТ** | 🟢 Высокая |
| **Кодирование (Довженко и др.)** | ❌ Без доказательств (плацебо) |
| **Кодирование (химическое, «торпеда»)** | 🔴 Опасно, может убить |

### Никотин

| Метод | Эффективность |
|---|---|
| **Никотинозамещающая терапия (пластырь, жвачка)** | +50-70% шанс бросить |
| **Варениклин (Champix)** | Самый эффективный препарат |
| **Электронные сигареты (меньший вред)** | На 95% менее вредны (Public Health England) |
| **Холодный отказ (Cold Turkey)** | ~5% успеха |
| **Когнитивная терапия** | +30% |

### Игровая / техно-зависимость

| Метод | Суть |
|---|---|
| **Цифровой детокс** | 24ч–7д без экранов |
| **Листы экранного времени** | Осознание (часто шокирует) |
| **Правило 20-20-20** | Каждые 20 мин — 20 сек вдаль |
| **Замена на физическую активность** | Спорт как альтернатива |

## 4. Срыв — не провал

- **80%** курильщиков срываются 3-5 раз перед успехом
- **Срыв ≠ рецидив**: один раз — срыв, возврат к регулярному — рецидив
- **Анализ**: что привело к срыву? Как избежать в следующий раз?
- **Без самобичевания**: стыд и вина → ещё один цикл

---

## 5. Когда нужен специалист

| Признак | Когда |
|---|---|
| Не можете бросить самостоятельно | После 3+ попыток |
| Абстиненция опасна | Алкоголь, бензодиазепины |
| Сопутствующая депрессия/тревога | Требует комплексного лечения |
| Риск для жизни | Передозировки, опасное поведение |
| Зависимость длится >5 лет | Профессиональная помощь обязательна |

## Практические советы

1. **Осознать триггеры** — вести дневник cravings (тяги)
2. **Убрать доступ** — алкоголь из дома, удалить приложения
3. **Добавить замену** — спорт, медитация, хобби
4. **Правило «Ни одного раза»** — не «только сегодня», а «никогда»
5. **Не пить/не курить «за компанию»** — научиться говорить нет
6. **Рассказать близким** — социальная поддержка критична
7. **Простить срывы** — каждый срыв — шаг к успеху, не поражение
