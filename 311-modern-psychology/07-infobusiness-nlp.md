# Инфобизнес и НЛП

> Анализ индустрии инфобизнеса, нейролингвистического программирования и манипуляций.

---

## 1. Инфобизнес: как устроен

### Бизнес-модель

```
Бесплатный контент (блог/Youtube/TikTok)
    ↓
Лид-магнит (бесплатная книга/вебинар)
    ↓
Прогрев (email, чат, сторис)
    ↓
Основной продукт ($50–500)
    ↓
Up-sell / Cross-sell ($500–5000)
    ↓
Наставничество / Коучинг ($5000–50000)
```

### Типичные обещания

| Обещание | Реальность |
|---|---|
| «Заработай $10 000 за месяц с нуля» | Зарабатывают те, кто продаёт этот курс |
| «Уникальная методика, скрытая от большинства» | Обычно компиляция из книг + харизма |
| «Мои ученики зарабатывают миллионы» | 1% учеников, остальные — нет |
| «Гарантия результата или деньги назад» | Гарантии почти никогда не выполняют |

### Как отличить полезный курс от инфоцыганщины

| ✅ Качественный курс | ❌ Инфоцыганщина |
|---|---|
| Автор имеет практический опыт | «Эксперт» без реального бэкграунда |
| Конкретные навыки (как сделать X) | «Успешный успех», мотивация |
| Трезвые обещания (через 6 мес.) | «$1M за неделю» |
| Есть отзывы о реальном результате | Отзывы только «спасибо за курс» |
| Можно ознакомиться бесплатно (часть) | Только платный доступ |
| Решаемая проблема | Размытые проблемы |

## 2. НЛП (Нейролингвистическое программирование)

### История

| Год | Событие |
|---|---|
| **1970-е** | Ричард Бендлер и Джон Гриндер изучают успешных терапевтов (Перлз, Сатир, Эриксон) |
| **1975** | «The Structure of Magic» — первая книга |
| **1980-е** | НЛП становится коммерческим продуктом |
| **1990+** | Раскол школы, критика, легитимизация отдельных техник |

### Основные концепции НЛП

| Концепция | Суть | Научность |
|---|---|---|
| **Раппорт** | Подстройка к собеседнику | 🟡 Есть элементы (отзеркаливание работает) |
| **Рефрейминг** | Смена контекста/значения | ✅ Есть в КПТ (когнитивное переструктурирование) |
| **Якоря** | Связь стимула с состоянием | 🟡 Классическое обусловливание |
| **Шкалы субъективного опыта** | Измерение интенсивности | ❌ Субъективно |
| **Субмодальности** | Детали внутренних образов | ❌ Не валидированы |
| **Мета-модель** | Вопросы для уточнения | 🟡 Полезный коммуникативный инструмент |
| **Взмах** | Замена триггерной реакции | 🟡 Частичные данные |

### Реальная эффективность НЛП

| Применение | Эффективность |
|---|---|
| Психотерапия (фобии, ПТСР) | 🟡 Ограниченные данные, уступает КПТ |
| Спорт, бизнес, коммуникация | 🟡 Плацебо / эффект новизны |
| Продажи (раппорт, подстройка) | 🟡 Есть элементы влияния |
| «Исцеление за один сеанс» (фобия) | 🔴 Преувеличено |

### Критика НЛП

| Критика | Обоснование |
|---|---|
| **Нет доказательной базы** | Мета-анализы: НЛП не работает лучше плацебо |
| **Псевдонаучная терминология** | «Якоря», «метапрограммы» — не операционализировано |
| **Самосбывающееся пророчество** | Создание «научного» фасада |
| **Маркетинг вместо науки** | Сертификация НЛП-практиков — бизнес, не наука |
| **Эклектика** | Смесь КПТ + гипноз + гештальт + эзотерика |

**Статус**: ❌ Псевдонаука (по классификации Beyerstein, Lilienfeld)

## 3. Манипуляции и защита

### Типичные манипулятивные приёмы

| Приём | Пример | Защита |
|---|---|---|
| **«Только сильные духом…»** | «Только сильные признают свои ошибки. Ты же сильный?» | Осознать рамку |
| **«Бесплатный сыр»** | Бесплатный вебинар → продажа курса | Не давать контакты |
| **«Авторитет»** | «Профессор такой-то рекомендует» | Проверить источник |
| **«Дефицит»** | «Осталось всего 3 места» | Эмоции vs рациональность |
| **«Социальное доказательство»** | «1000+ учеников уже купили» | Отзывы могут быть липовыми |
| **«Взаимность»** | Бесплатный подарок → ожидание услуги | Просто сказать нет |

## 4. Полезные элементы (без эзотерики)

Из НЛП и инфобизнеса можно взять:

1. **Постановка целей (SMART + Well-Formed Outcomes)** — работает
2. **Мета-модель** — полезный инструмент для уточнения мыслей
3. **Раппорт (отзеркаливание)** — работает в переговорах
4. **Позитивное намерение** — искать конструктивное в поведении
5. **Фрейминг** — навык менять контекст ситуации

**Но**: всё это есть в КПТ, коммуникативной психологии и переговорных техниках (Гарвардская школа) — с лучшей доказательной базой.
