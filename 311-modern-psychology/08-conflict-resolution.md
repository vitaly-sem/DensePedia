# Как правильно разрешать конфликты

> Практические техники и стратегии для конструктивного разрешения конфликтов.

---

## 1. Природа конфликта

### Пирамида конфликта

```
Отношения (накопленные обиды)
    ↓
Интересы (что каждая сторона хочет)
    ↓
Позиции (что каждая сторона говорит)
```

### Типы конфликтов

| Тип | Пример | Стратегия |
|---|---|---|
| **Интересов** | Делим бюджет | Компромисс / интеграция |
| **Ценностей** | Религия, политика | Согласие на несогласие |
| **Отношений** | Личные обиды | Восстановление доверия |
| **Структурный** | Роли, ресурсы | Изменение системы |

---

## 2. Техники разрешения

### Гарвардская модель переговоров (Fisher, Ury)

**4 принципа**:

1. **Люди**: отделяйте людей от проблемы
2. **Интересы**: фокус на интересах, а не на позициях
3. **Варианты**: ищите взаимовыгодные варианты
4. **Критерии**: используйте объективные критерии

### Техника DESC (Assertiveness)

```
D — Describe: описать ситуацию без оценки
  «Когда ты опаздываешь на встречи…»
E — Express: выразить чувства
  «…я чувствую неуважение»
S — Specify: предложить решение
  «…можешь предупреждать за 30 минут»
C — Consequence: указать последствия
  «…иначе я буду планировать без тебя»
```

### Техника тайм-аута

```
1. Чувствуете гнев → скажите «Мне нужно 20 минут»
2. Уйдите в другую комнату
3. Дышите медленно (вдох 4 сек, выдох 6 сек)
4. Вернитесь → продолжите разговор спокойно
```

### S.T.O.P.

```
S — Stop (остановиться)
T — Take a breath (дышать)
O — Observe (наблюдать чувства)
P — Proceed (продолжить осознанно)
```

## 3. Фазы разрешения

| Фаза | Что делать |
|---|---|
| **1. Разрядка** | Дать эмоциям остыть (10–20 мин) |
| **2. Выслушать** | Активное слушание, не перебивать |
| **3. Валидировать** | «Я понимаю, почему ты злишься» |
| **4. Назвать интересы** | Чего я хочу на самом деле? Чего хочет он? |
| **5. Генерация вариантов** | Мозговой штурм без критики |
| **6. Оценка вариантов** | Какой вариант выгоден обеим сторонам? |
| **7. Договорённость** | Чёткие договорённости, сроки |
| **8. Проверка** | Через неделю: работает ли? |

---

## 4. Чего не делать

| ❌ Не делайте | Почему |
|---|---|
| **Кричать / оскорблять** | Активирует защиту, прекращает диалог |
| **Вспоминать прошлое** | «А помнишь, ты тогда…» → эскалация |
| **Генерализировать** | «Ты всегда…», «Ты никогда…» |
| **Молчать / игнорировать** | Каменная стена → разрыв |
| **Угрожать** | «Если ты не… то я…» |
| **Сравнивать** | «А вот у других…» |
| **Драматизировать** | «Ты меня больше не любишь» |
| **Решать за другого** | «Ты должен…» |

---

## 5. Конфликты на работе

| Ситуация | Подход |
|---|---|
| **С коллегой** | Личная беседа, не вовлекать других |
| **С начальником** | Факты, предложения, не эмоции |
| **С подчинённым** | Конструктивная обратная связь |
| **В команде** | Фасилитация, медиация |
| **С клиентом** | Сервисное мышление, win-win |

### Модель «Трудный разговор» (Patterson)

1. **Сделайте разговор безопасным** — «Моя цель — не обвинить, а понять»
2. **Поделитесь фактами** — «Вчера на совещании ты сказал…»
3. **Расскажите о последствиях** — «Я почувствовал…»
4. **Попросите их версию**
5. **Найдите решение вместе**

---

## 6. Когда конфликт не решается

| Причина | Действие |
|---|---|
| **Разные ценности** | Согласиться на несогласие |
| **Эмоциональная незрелость** | Установить границы |
| **Токсичность** | Минимизировать контакт |
| **Отсутствие желания** | Нельзя решить конфликт в одиночку |
| **Нужна помощь** | Медиатор, психолог, HR |

---

## 7. Профилактика конфликтов

- **Честность с самого начала** — недоговорённости → конфликты
- **Еженедельные чекины** — как дела, что беспокоит
- **Регулярная обратная связь** — не копить обиды
- **Культура извинений** — «Извини» + конкретное изменение поведения
- **Общие цели** — напоминать, что мы в одной команде
