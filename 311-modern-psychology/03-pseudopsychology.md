# Псевдопсихологи и за что они цепляют

> Критика популярных псевдопсихологов, шарлатанов и их методов.

---

## 1. Графология (почерк → характер)

| Утверждение | Научный ответ |
|---|---|
| «По почерку можно определить характер» | Мета-анализы: корреляция ~0.02 (нулевая) |
| «Почерк меняется с настроением» | Да, но это не диагностика характера |
| «Используется при найме» | В большинстве стран запрещено как дискриминация |

**Статус**: ❌ Псевдонаука

---

## 2. Соционика и MBTI

### MBTI (Myers-Briggs Type Indicator)

| Утверждение | Научный ответ |
|---|---|
| «16 типов личности» | Big Five — научная альтернатива (OCEAN) |
| «Тест надёжен» | 50% людей получают другой тип через 5 недель |
| «Используется в компаниях» | Да, но научная психология критикует |

**Статус**: ❌ Невалиден (нет предсказательной силы)

### Соционика (Аугустинавичюте)

| Проблема | Обоснование |
|---|---|
| Основана на MBTI (сам по себе спорный) | Нет рецензируемых исследований |
| 16 типов с интертипными отношениями | Нет корреляции с реальным поведением |
| Популярна на русскоязычном пространстве | Нет признания академической психологией |

**Статус**: ❌ Псевдонаука

---

## 3. Психолог vs Коуч vs Шарлатан

### Красные флаги псевдопсихолога

| Флаг | Почему это плохо |
|---|---|
| **«Уникальная авторская методика»** | Если работает — почему не опубликована в рецензируемых журналах? |
| **«Гарантирую результат за 3 сессии»** | КПТ — 6-20 сессий. Быстрое «исцеление» = обман |
| **«Все проблемы из прошлой жизни»** | Не тестируемо → не наука |
| **«Все проблемы из детства»** | Полуправда: важно, но не единственная причина |
| **«Работа с родом» (расстановки)** | Расстановки Хеллингера — спорный метод |
| **«Диагноз по фото»** | Шарлатанство чистой воды |
| **«Исцеление без контакта»** | Не наука |
| **«Психосоматика всего»** | Психосоматика существует, но не всё — психосоматика |

## 5. Почему люди верят?

| Причина | Объяснение |
|---|---|
| **Эффект Барнума (Форера)** | Общие описания кажутся личными |
| **Подтверждающее искажение** | Ищем подтверждение, игнорируем опровержения |
| **Желание быстрых решений** | Терапия требует времени и усилий |
| **Незнание научной психологии** | Шарлатаны используют наукообразные слова |
| **Культ авторитета** | Яркая личность → доверие |

## 6. Как отличить настоящего психолога

| ✅ Настоящий психолог | ❌ Шарлатан |
|---|---|
| Имеет профильное образование (ВУЗ) | Сертификаты 3-дневных курсов |
| Использует методы с доказанной эффективностью (КПТ, DBT, ACT) | «Авторская методика» |
| Не даёт гарантий | Гарантирует излечение |
| Не ставит диагнозов (если не психиатр) | Ставит «диагнозы» по фото |
| Работает в этических рамках | Нарушает границы |
| При проблемах направляет к психиатру | Отрицает медицину |
| Соблюдает конфиденциальность | Публикует «кейсы» с клиентами |
