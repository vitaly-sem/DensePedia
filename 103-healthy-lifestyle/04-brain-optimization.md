# Работа мозга — когнитивная оптимизация

---

## Нейрофизиология — основы

**Факт:** Мозг потребляет **20% всей энергии** организма при 2% веса (~350 ккал/день).

**Факт:** **Нейропластичность** — мозг меняется под воздействием опыта в любом возрасте. Не «застывает» после 25.

**Факт:** **BDNF** (Brain-Derived Neurotrophic Factor) — белок, стимулирующий рост новых нейронов и синапсов. Повышается от: аэробных упражнений, интервального голодания, омега-3.

---

## Нейротрансмиттеры

| Нейротрансмиттер | Функция | Как повысить | Дефицит |
|---|---|---|---|
| **Дофамин** | Мотивация, удовольствие, фокус | Достижение целей, холодный душ, спорт | Апатия, прокрастинация |
| **Серотонин** | Настроение, сон, аппетит | Солнце (15 мин), спорт, социальные контакты | Депрессия, тревога |
| **Норадреналин** | Бдительность, реакция | Холод, стресс (кратковременный), кофеин | Усталость |
| **Ацетилхолин** | Обучение, память | Яйца, рыба, ноотропы (альфа-GPC) | Плохая память |
| **GABA** | Торможение, спокойствие | Медитация, магний, L-теанин | Тревога, бессонница |
| **Глутамат** | Возбуждение, обучение | — | Избыток = нейротоксичность |

## Практические протоколы

### Для фокуса (утро)
```
07:00 — Холодный душ 2-3 мин (↑ дофамин + норадреналин)
07:30 — Кофе 100 мг + L-теанин 200 мг
08:00 — Deep work (90 мин, без телефона)
10:00 — Прогулка 15 мин (естественный свет)
```

### Для памяти (обучение)
```
Spaced repetition (Anki/RemNote) — повторение через увеличивающиеся интервалы
Active recall — вспоминать без подсказок
Interleaving — чередовать темы
Feynman technique — объяснить сложное простыми словами
```

### Для сна (вечер)
```
21:00 — Без экранов (синий свет ↓ мелатонин)
21:30 — Чтение (бумажная книга)
22:00 — Магний 200 мг + глицин 3 г
22:30 — Сон при 18°C, полная темнота
```

## Факторы, ухудшающие когнитивные функции

| Фактор | Механизм | Эффект |
|---|---|---|
| **Недостаток сна** | Глимфатическая система не очищает β-амилоид | Накопление токсинов, риски нейродегенерации |
| **Стресс (хронический)** | Кортизол → атрофия гиппокампа | Ухудшение памяти |
| **Сахар (высокий ГИ)** | Инсулин → воспаление мозга | «Brain fog» |
| **Алкоголь** | Нейротоксин, убивает нейроны | Атрофия мозга (даже «умеренное» потребление) |
| **Малоподвижность** | ↓ BDNF, ↓ кровоток мозга | Когнитивный спад |
| **Изоляция** | ↓ Социальная стимуляция | Деменция |
| **Информационный шум** | Дофаминовая перегрузка | ↓ Фокус, ↓ глубина мышления |

- [ ] Аэробная нагрузка 150 мин/нед (↑ BDNF)

- [ ] Омега-3 (жирная рыба 2-3 раза/нед)
- [ ] Холодный душ (↑ дофамин на 250% на 1-2 ч)
- [ ] Медитация (10 мин/день, ↑ серотонин, ↓ кортизол)
- [ ] Social connections (протектор от деменции)
- [ ] Lifelong learning (нейропластичность)
