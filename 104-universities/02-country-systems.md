# Национальные системы образования

---

## США

**Факт:** В США нет единой федеральной системы высшего образования — каждый штат регулирует самостоятельно.

| Характеристика | Описание |
|---|---|
| **Степени** | Associate (2 года) → Bachelor (4 года) → Master (1–2 года) → PhD (5–7 лет) |
| **Типы вузов** | Community Colleges, Public State Universities, Private Universities, Liberal Arts Colleges |
| **Liberal Arts** | Первые 2 года — общее образование (general education), специализация — junior/senior годы |
| **Grading** | GPA (0.0–4.0), A–F, ± система |
| **Tuition** | $10K–$60K/год; public для резидентов штата — дешевле |
| **Финансирование** | Federal/State aid, scholarships, student loans (~$1.7 трлн общий долг) |
| **Адмиссии** | SAT/ACT, GPA, extracurriculares, essays, recommendation letters |
| **Календарь** | Semester (fall/spring) или Quarter system |

**Ключевые группы:**
- **Ivy League** (8 частных: Harvard, Yale, Princeton, Columbia, Brown, Dartmouth, UPenn, Cornell) — элита, ~5% acceptance rate
- **Public Ivies** (UCLA, UC Berkeley, Michigan, UVA, UNC) — топ-публичные
- **Little Ivies** (Amherst, Williams, Swarthmore) — элитные Liberal Arts Colleges

## Великобритания

**Факт:** Оксфорд и Кембридж (Oxbridge) используют **коллегиальную систему** — студент принадлежит конкретному колледжу.

| Характеристика | Описание |
|---|---|
| **Степени** | Foundation (1 год) → Bachelor (3 года, Шотландия — 4) → Master (1 год) → PhD (3–4 года) |
| **Типы вузов** | Ancient (до 1600), Red Brick (XIX век), Plate Glass (1960-е), Post-1992 (polytechnics) |
| **Russell Group** | 24 ведущих исследовательских университета (Oxford, Cambridge, Imperial, LSE, UCL, Edinburgh) |
| **Grading** | First (70%+), Upper Second (60–69%), Lower Second (50–59%), Third (40–49%) |
| **Tuition** | Home: ~£9,250/год; International: £15K–£50K/год |
| **Адмиссии** | A-levels (3 предмета), Personal Statement, Interview (Oxbridge), LNAT/BMAT |
| **Особенности** | Глубокая специализация с 1-го курса (3 предмета A-level → 1 специальность) |

**Факт:** В Англии Bachelor — **3 года** (против 4 в США и Шотландии). Master — **1 год** (против 2 лет в США).

## Германия

**Факт:** Большинство немецких университетов — **государственные** и **бесплатные** (даже для иностранцев, с семестровым взносом ~€300).

| Характеристика | Описание |
|---|---|
| **Степени** | Bachelor (3–4 года) → Master (1–2 года) → PhD (3–5 лет) |
| **Типы вузов** | Universität (фундаментальная наука), Fachhochschule / HAW (прикладные), Kunst-/Musikhochschulen |
| **TU9** | 9 ведущих технических университетов (TUM, RWTH Aachen, KIT, TU Berlin, TU Dresden) |
| **Grading** | 1.0 (sehr gut) – 5.0 (nicht bestanden) |
| **Tuition** | Semesterbeitrag €150–€400; Baden-Württemberg — €1,500/семестр для не-EU |
| **Адмиссии** | Abitur (аттестат), NC (Numerus Clausus) для популярных специальностей |
| **Дуальное обучение** | Теория в вузе + работа в компании, зарплата от €800/мес |

**Факт:** **Fachhochschulen** — уникальный немецкий тип: прикладные науки, обязательный практический семестр, меньший research, но высокая трудоустраиваемость.

## Франция

**Факт:** Французская система — **двойная**: университеты (открытый доступ) vs Grandes Écoles (элитные, конкурсный отбор).

| Характеристика | Описание |
|---|---|
| **Степени** | Licence (3 года) → Master (2 года) → Doctorat (3 года) |
| **Grandes Écoles** | École Polytechnique, CentraleSupélec, HEC, ESSEC, Sciences Po — элита, prépa (2 года) + concours |
| **Университеты** | Sorbonne, PSL, Paris-Saclay — массовое образование, низкая стоимость |
| **Grading** | 0–20; 10/20 — проходной, 16+ — исключительный (редко) |
| **Tuition** | Université: €170–€380/год (Licence); Grandes Écoles: €3K–€15K/год |
| **Особенности** | Централизованное министерство, национальные дипломы |

**Факт:** Во Франции оценка **14/20 — это «отлично»**, тогда как для американской системы это ~70% (C). Культурные различия в grading — огромны.

## Россия

| Характеристика | Описание |
|---|---|
| **Степени** | Бакалавр (4 года) → Магистр (2 года) → Аспирант (3–4 года); Специалитет (5–6 лет — медицина, инженерия) |
| **Типы вузов** | Федеральные (МГУ, СПбГУ), Национальные исследовательские (ВШЭ, МФТИ, МИФИ, ИТМО), Региональные |
| **5-100 / Приоритет 2030** | Программа повышения конкурентоспособности российских вузов |
| **ЕГЭ** | Единый государственный экзамен — основной критерий поступления |
| **Grading** | 2 (неуд.) – 3 – 4 – 5 (отл.); зачёт/незачёт |
| **Tuition** | Бюджет (бесплатно, по конкурсу) или контракт (платно, от ~50K до ~800K ₽/год) |
| **Особенности** | СССР: специалитет, жёсткая программа, идеология. Современная Россия: переход на Болонскую систему (но частичный откат с 2024) |

## Китай

**Факт:** Китай — крупнейшая система высшего образования в мире (>40 млн студентов).

| Характеристика | Описание |
|---|---|
| **Степени** | Bachelor (4 года) → Master (2–3 года) → PhD (3–4 года) |
| **C9 League** | 9 элитных университетов (Tsinghua, Peking, Fudan, Shanghai Jiao Tong, Zhejiang, Nanjing, USTC, HIT, Xi'an Jiaotong) |
| **Gaokao** | Национальный вступительный экзамен — один из самых сложных в мире |
| **Grading** | 4.0 GPA или 100-балльная шкала |
| **Tuition** | ~¥5K–¥15K/год (домашние); выше для международных |
| **Особенности** | 985/211 проекты — государственные инвестиции в топ-вузы; рост research output |

**Факт:** Gaokao сдаётся **один раз в год** (июнь), длится 2–3 дня, 9+ часов экзаменов. Результат — единственный критерий поступления.

## Япония

| Характеристика | Описание |
|---|---|
| **Степени** | Bachelor (4 года) → Master (2 года) → PhD (3 года) |
| **Типы вузов** | Национальные (Tokyo, Kyoto, Tohoku, Osaka), Публичные (префектуральные), Частные (Waseda, Keio) |
| **Top 7 (Imperial Universities)** | Tokyo, Kyoto, Tohoku, Kyushu, Hokkaido, Osaka, Nagoya |
| **Вступительные** | Center Test + индивидуальные экзамены каждого вуза |
| **Grading** | A–F, 90–100% = A, или 4.0 GPA |
| **Tuition** | Национальные: ~¥535,800/год (~$3,600); Частные: ~¥1M–¥1.5M/год |
| **Особенности** | Культура «экзаменационного ада» (shiken jigoku); пожизненный найм — влияние на выбор вуза |
