# Как защитить свои данные от хакеров и мошенников

> Практическое руководство по защите личных данных — от паролей до безопасности устройств.

---

## 1. Пароли

### Как не надо

| ❌ Плохо | Почему |
|---|---|
| `123456`, `password`, `qwerty` | Первое, что проверяют |
| Дата рождения, имя ребёнка | Легко узнать из соцсетей |
| Один пароль везде | Утечка одного = утечка всех |
| Пароль на стикере на мониторе | Физический доступ |

### Как надо

| ✅ Хорошо | Как сделать |
|---|---|
| **Длинный** (>12 символов) | `mYc4t!sfluffy&l0vesF1sh` |
| **Уникальный** для каждого сервиса | Менеджер паролей |
| **Символы** (буквы + цифры + спецсимволы) | A-Z, a-z, 0-9, !@#$% |
| **Не словарный** | Не `iloveyou`, а `G7!kP9#mQ2$w` |

### Менеджеры паролей

| Сервис | Цена | Особенность |
|---|---|---|
| **Bitwarden** | Бесплатно (базовый) | Open source, все платформы |
| **1Password** | $3/мес | Семейные планы |
| **KeePass** | Бесплатно | Локальное хранение |
| **iCloud Keychain** | Бесплатно (Apple) | Встроен в macOS/iOS |
| **Google Password Manager** | Бесплатно | Встроен в Chrome/Android |

## 2. Двухфакторная аутентификация (2FA)

### Что лучше

| Тип | Безопасность | Удобство |
|---|---|---|
| **SMS-код** | 🟡 Средняя (SIM-swap) | Высокое |
| **Приложение (Google Authenticator, Authy)** | 🟢 Высокая | Среднее |
| **Физический ключ (YubiKey, Titan)** | 🟢🔝 Максимальная | Низкое |
| **Биометрия (палец, лицо)** | 🟡 Средняя | Высокое |
| **Push-уведомление** | 🟡 Средняя | Высокое |

**Правило**: включите 2FA везде, где возможно. Начните с почты, банка, соцсетей.

## 3. Фишинг — главная угроза

### Как распознать

| Признак | Пример |
|---|---|
| **Спешка** | «Ваш аккаунт будет заблокирован через 24 часа» |
| **Ссылка не туда** | `g00gle.com` вместо `google.com` |
| **Грамматические ошибки** | «Уважаемый клиент, вам сообщение из банк» |
| **Запрос данных** | Банк никогда не просит CVV/пароль по email |
| **Неожиданное вложение** | `счёт.pdf.exe` |
| **«Вы выиграли»** | Незнакомые лотереи / наследства |

### Что делать

1. **Не переходить** по ссылке из письма
2. **Навести курсор** — посмотреть реальный URL
3. **Зайти в сервис напрямую** — набрать в браузере самим
4. **Проверить email** — отправитель, подпись DKIM/SPF
5. **Сообщить** — в IT-отдел (работа) или провайдеру

## 4. Безопасный сёрфинг

### Браузер

| Мера | Как включить |
|---|---|
| **HTTPS-only** | Включить в Chrome/Safari/Firefox |
| **Блокировка трекеров** | DuckDuckGo, Firefox Enhanced Tracking, Brave |
| **Антифишинг** | Встроен в Chrome/Safari/Edge |
| **Расширения** | uBlock Origin, Privacy Badger, HTTPS Everywhere |
| **Не сохранять пароли** | Лучше менеджер паролей |
| **VPN** | При публичном Wi-Fi (не бесплатные — они продают данные) |

### Соцсети

- **Закрыть профиль** — не публичный
- **Не публиковать** — адрес, телефон, паспорт, билеты, геолокацию
- **Проверить** — какие приложения имеют доступ к аккаунту (удалить лишние)
- **Двухфакторка** — обязательно

## 5. Защита устройств

### Смартфон

| Мера | Важность |
|---|---|
| **Блокировка (пин/биометрия)** | 🔴 Обязательно |
| **Автообновление ОС** | 🔴 Обязательно |
| **Не устанавливать из неофициальных источников** | 🔴 Обязательно |
| **Не jailbreak/root** | 🔴 Категорически |
| **Удалять неиспользуемые приложения** | 🟡 Желательно |
| **Проверить разрешения приложений** | 🟡 Желательно |

### Компьютер

- **Антивирус**: Windows Defender (встроен — достаточно) + Malwarebytes
- **Обновления**: включить автоматические (критические уязвимости)
- **Бэкапы**: правило 3-2-1 (3 копии, 2 носителя, 1 удалённый)
- **Шифрование диска**: BitLocker (Windows), FileVault (macOS)
- **Не сидеть под админом**: отдельный пользователь без прав admin

## 6. Защита от телефонного мошенничества

### Схемы

| Схема | Как распознать |
|---|---|
| **«Звонок из банка»** | Банк не звонит сам, не просит назвать CVV/пароль. Положить трубку |
| **«Ваш родственник попал в ДТП»** | «Нужны деньги на операцию адвокату». Положить трубку |
| **«Сотрудник полиции/ФСБ»** | Настоящие силовики не звонят для «проверки счетов» |
| **«Выгодное предложение от брокера»** | Не существует гарантированного дохода |
| **«Ваш номер заблокируют»** | Операторы связи так не делают |

**Главное правило**: 
- Незнакомец звонит и говорит о деньгах — **кладите трубку**.
- Никто не имеет права требовать перевод на «безопасный счёт».
- Перезвоните в банк/полицию/оператору **сами** по официальному номеру.

## 7. Социальная инженерия

### Как манипулируют

| Приём | Пример |
|---|---|
| **Авторитет** | «Я из службы безопасности, ваш аккаунт взломан» |
| **Срочность** | «Действуйте немедленно, иначе деньги пропадут» |
| **Взаимность** | «Я вам помог, теперь помогите мне — скажите код» |
| **Симпатия** | «Вы такой приятный человек, помогите с настройкой» |

### Защита

- Никому **не сообщайте** SMS-коды, CVV, PIN
- Никто из банка/полиции **не попросит** перевод на счёт
- Если собеседник давит и торопит — **стоп**. Мошенники давят всегда
- **Правило «30 секунд»**: если незнакомец просит что-то сделать за 30 сек — это обман

## 8. Если уже взломали

| Ситуация | Действие |
|---|---|
| **Угнали аккаунт** | Пароль → восстановить через почту/телефон. Проверить привязанные устройства |
| **Взломали почту** | Сменить пароль → проверить пересылку писем → 2FA |
| **Потеряли телефон** | Заблокировать SIM → включить Find My → удалённо стереть |
| **Украли данные карты** | Заблокировать карту → перевыпустить |
| **Попался на фишинг** | Сменить пароли → включить 2FA → проверить аккаунты |
