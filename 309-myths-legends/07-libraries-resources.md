## Практические советы для исследователя

1. **Начинайте с Theoi.com** для греческой мифологии — лучшая база
2. **Sacred-texts.com** — для первоисточников всех культур
3. **Кэмпбелл «Тысячеликий герой»** — для понимания структуры мифа
4. **Леви-Стросс «Мифологики»** — для анализа мифов как системы знаков
5. **Избегайте непроверенных интернет-источников** — многие «мифологические» факты вымышлены
