# Научные теории происхождения

> Теории, признанные научным сообществом, имеющие эмпирические подтверждения

## 1. Теория Большого взрыва (происхождение Вселенной)

| Параметр | Значение |
|---|---|
| **Автор** | Жорж Леметр (1927), подтверждено Хабблом (1929) |
| **Статус** | Доминирующая научная теория |
| **Суть** | Вселенная возникла из сингулярности ~13.8 млрд лет назад |

### Доказательства

| Доказательство | Когда открыто | Суть |
|---|---|---|
| **Расширение Вселенной (закон Хаббла)** | 1929 | Галактики разбегаются — Вселенная расширяется |
| **Реликтовое излучение (CMB)** | 1965 (Пензиас, Вильсон) | «Эхо» Большого взрыва, ~2.7K |
| **Распространённость лёгких элементов** | 1940-е | H (75%), He (24%), Li — соотношение, предсказанное БВ |
| **Крупномасштабная структура** | 2000-е | Распределение галактик соответствует моделям БВ |

### Ограничения

- **Сингулярность**: физика не работает при t=0. Теории квантовой гравитации пока нет
- **Тёмная материя (~27%)** и **тёмная энергия (~68%)**: не наблюдаются напрямую

## 2. Абиогенез (происхождение жизни)

| Параметр | Значение |
|---|---|
| **Идея** | Жизнь возникла из неживой материи через химические реакции |
| **Время** | ~3.5–4 млрд лет назад |
| **Статус** | Подтверждённая гипотеза (механизмы уточняются) |

### Ключевые эксперименты и доказательства

| Эксперимент/Открытие | Год | Суть |
|---|---|---|
| **Опыт Миллера — Юри** | 1953 | Аминокислоты из газов (CH₄, NH₃, H₂, H₂O) + электричество |
| **Рибозим (Т. Чек, С. Олтман)** | 1982 | РНК может быть и информацией, и катализатором (мир РНК) |
| **Гидротермальные источники** | 1977 | Хемосинтез — жизнь без солнца |
| **Метеорит Мёрчисон** | 1969 | Аминокислоты и нуклеотиды в космосе |
| **Протоклетки (Сепик, Димер)** | 2000-е | Самосборка липидных мембран, содержащих РНК |

### Этапы (гипотеза мира РНК)

1. **Химическая эволюция**: простые молекулы → аминокислоты, нуклеотиды
2. **РНК-мир**: РНК — первые самовоспроизводящиеся молекулы
3. **Протоклетки**: мембрана + РНК
4. **Последний универсальный общий предок (LUCA)**: ~3.5 млрд лет назад

## 3. Эволюция (происхождение видов)

| Параметр | Значение |
|---|---|
| **Авторы** | Чарльз Дарвин (1859), Альфред Уоллес |
| **Статус** | Основополагающая теория биологии |

### Доказательства

| Доказательство | Примеры |
|---|---|
| **Ископаемые** | Переходные формы: тиктаалик (рыба→четвероногие), археоптерикс (динозавр→птица) |
| **Гомологичные органы** | Конечности: рука человека, лапа кошки, крыло летучей мыши, плавник кита |
| **Рудименты** | Аппендикс, копчик, зубы мудрости, мышцы ушной раковины |
| **ДНК-свидетельства** | 98.5% совпадения с шимпанзе, 60% с бананом |
| **Эволюция в действии** | Устойчивость бактерий к антибиотикам, галапагосские вьюрки (Грант, 1970-е) |
| **Псевдогены** | «Сломанные» гены, общие для человека и других приматов (GULO — витамин C) |

### Механизм

```
Мутация → Вариация → Естественный отбор → Адаптация
  (случайная)    (разнообразие)  (выживание лучших)    (изменение)
```

## 4. Панспермия (происхождение жизни из космоса)

| Параметр | Значение |
|---|---|
| **Идея** | Жизнь занесена на Землю из космоса (метеориты, кометы) |
| **Статус** | Возможна, но отодвигает вопрос происхождения (откуда жизнь в космосе?) |

### Доказательства

| Факт | Значение |
|---|---|
| **Радиорезистентность Deinococcus radiodurans** | Бактерии выживают в открытом космосе |
| **Органика в метеоритах** | Аминокислоты, нуклеотиды в метеоритах |
| **Эксперимент BIOPAN (МКС)** | Лишайники выжили в космосе |
| **Mars meteorite ALH84001** | Спорно: возможные микроокаменелости |

### Проблемы

- **Выживание в космосе**: радиация, ультрафиолет, вакуум
- **Происхождение жизни**: если жизнь занесена — откуда она там?

## 5. Современный синтез (происхождение человека)

| Параметр | Значение |
|---|---|
| **Вид** | Homo sapiens (200 000 лет) |
| **Теория** | «Из Африки» (Out of Africa, ~60 000 лет назад) |

### Эволюционная линия человека

| Вид | Период | Объём мозга | Особенность |
|---|---|---|---|
| **Sahelanthropus tchadensis** | ~7 млн лет | 350 cm³ | Возможно, первый гоминид |
| **Australopithecus afarensis (Люси)** | ~3.9-2.9 млн | 400 cm³ | Прямохождение |
| **Homo habilis** | ~2.8-1.5 млн | 600 cm³ | Первые каменные орудия |
| **Homo erectus** | ~2 млн-100 тыс | 900 cm³ | Огонь, миграция из Африки |
| **Homo neanderthalensis** | ~400-40 тыс | 1500 cm³ | Культура, захоронения |
| **Homo sapiens** | ~300 тыс — н.в. | 1350 cm³ | Язык, искусство, сельское хозяйство |

### Доказательства

- **Ископаемые**: Люси (1974, Эфиопия), Турканский мальчик (1984, Кения)
- **Палеогенетика**: секвенирование неандертальца (2010) — 1-4% ДНК у неафриканцев
- **Митохондриальная Ева**: ~150 000 лет, одна женщина (но не единственная женщина того времени)
- **Y-хромосомный Адам**: ~200 000 лет
