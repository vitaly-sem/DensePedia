# Псевдонаучные и альтернативные теории происхождения

> Теории, не признанные научным сообществом. Описание обоснований с указанием научной критики.

---

## 1. Креационизм / Младоземельный креационизм

| Параметр | Значение |
|---|---|
| **Идея** | Мир создан Богом за 6 буквальных дней, ~6000–10000 лет назад |
| **Основа** | Буквальное толкование Библии (Книга Бытия) |
| **Статус** | ❌ Отвергается наукой |

### Обоснование (сторонников)

- **Библия** как исторический источник
- **Отсутствие переходных форм** (аргумент: «пробел в летописи окаменелостей»)
- **Сложность жизни** требует Творца (аргумент: «часы требуют часовщика» — Уильям Пейли, 1802)
- **Второй закон термодинамики** — энтропия растёт, значит эволюция невозможна (неверное применение)
- **Радиометрическое датирование** — «ненадёжно» (опровергнуто)

### Научная критика

| Аргумент креационистов | Научный ответ |
|---|---|
| Миру 6000 лет | Дендрохронология: кольца деревьев >9000 лет. Ледяные керны: >800 000 лет |
| Нет переходных форм | Тиктаалик, археоптерикс, амбулоцет, эораптор — тысячи ископаемых |
| Сложность = разумный замысел | Эволюция может создавать сложность постепенно (глаз — 40 шагов) |
| Второй закон термодинамики | Земля — открытая система, Солнце даёт энергию |
| Потоп был глобальным | Нет геологических доказательств глобального потопа. Ковчег физически невозможен |

### Формы креационизма

| Тип | Суть |
|---|---|
| **Младоземельный** | 6000 лет, буквальная Библия |
| **Староземельный** | Миллиарды лет, но Бог направлял |
| **Теистическая эволюция** | Эволюция — инструмент Бога |
| **Разумный замысел (Intelligent Design)** | Научно выглядящая версия, без упоминания Бога |

## 2. Разумный замысел (Intelligent Design)

| Параметр | Значение |
|---|---|
| **Идея** | Некоторые структуры (жгутик бактерии, глаз) «несократимо сложны» — не могли возникнуть эволюцией |
| **Продвигается** | Discovery Institute (США) |
| **Статус** | ❌ Признан псевдонаукой (Kitzmiller v. Dover, 2005) |

### Обоснование

- **Несократимая сложность** (Майкл Бихи): бактериальный жгутик — все части обязательны
- **Специфицированная сложность** (Уильям Дембски): неслучайные паттерны требуют Творца

### Научная критика

- Бактериальный жгутик эволюционировал из секреторной системы (тип III — есть у бактерий без жгутика)
- «Специфицированная сложность» Дембски не имеет эмпирического применения
- Kitzmiller v. Dover (2005): суд постановил, что ID — религия, не наука

## 3. Древние астронавты (теория палеоконтакта)

| Параметр | Значение |
|---|---|
| **Идея** | Древние цивилизации посещали инопланетяне, которые передали технологии |
| **Популяризаторы** | Эрих фон Дэникен («Колесницы богов?», 1968), Захария Ситчин |
| **Статус** | ❌ Псевдоархеология |

### Обоснование (сторонников)

- **Пирамиды Египта** — «невозможно построить без пришельцев»
- **Линии Наска** — «посадочные полосы для НЛО»
- **Библия**: «колёса внутри колёс» (Иезекииль) — космический корабль
- **Остров Пасхи**: статуи моаи — «инопланетная технология»
- **Шумерские таблички**: аннунаки (Ситчин) — астронавты с Нибиру

### Научная критика

| Аргумент | Ответ |
|---|---|
| Пирамиды — слишком сложные | Доказано: построены людьми, египтяне оставили записи процесса |
| Линии Наска — посадочные полосы | Геоглифы видны с земли (ритуальные тропы) |
| Моаи — невозможно переместить | Экспериментально доказано: 18 человек могут перемещать |
| Ситчин перевёл шумерские тексты | Ассириологи: переводы Ситчина некорректны, аннунаки — «те, кто с небес сошли» (мифология) |

## 4. Плоская Земля

| Параметр | Значение |
|---|---|
| **Идея** | Земля плоская (диск), Антарктида — ледяная стена |
| **Популяризация** | Samuel Rowbotham (XIX в.), современный Flat Earth Society |
| **Статус** | ❌ Полностью опровергнута |

### Обоснование (сторонников)

- **Горизонт** кажется плоским
- **NASA врёт** — все снимки Земли подделка
- **Солнце и Луна** — маленькие, движутся над диском

### Научная критика

| Доказательство шарообразности | Когда |
|---|---|
| Корабли исчезают за горизонтом (корпус — раньше мачт) | Древняя Греция |
| Тень Земли на Луне (затмения) — круглая | Всегда |
| Разные созвездия в северном/южном полушарии | Эратосфен (III в. до н.э.) |
| Фотографии из космоса (первые — 1946, 1968) | XX век |
| Кругосветные путешествия | Магеллан (1522) |
| Спутниковая связь, GPS, авиаперелёты | XX–XXI вв. |

## 5. Геоцентризм (Земля — центр Вселенной)

| Параметр | Значение |
|---|---|
| **Идея** | Солнце и планеты вращаются вокруг Земли |
| **Статус** | ❌ Опровергнута Коперником, Галилеем, Кеплером |

### Обоснование (сторонников)

- Субъективно: Солнце «встаёт» и «садится»
- Библейские стихи (Иисус Навин 10:13 — «Солнце остановилось»)

### Научная критика

- Галилей (1610): фазы Венеры — невозможны при геоцентризме
- Параллакс звёзд (XIX в.): Земля движется вокруг Солнца
- Современная астрономия: гелиоцентризм полностью подтверждён
