# Платформы для создания и продажи плагинов

> Маркетплейсы, экосистемы и платформы, где разработчики могут создавать плагины/расширения и зарабатывать.

---

## 1. Браузерные расширения

### Chrome Web Store

| Параметр | Значение |
|---|---|
| **Аудитория** | 3+ млрд пользователей (Chrome) |
| **Язык** | JavaScript (Manifest V3) |
| **Регистрация** | $5 (однократный взнос) |
| **Доля платформы** | 30% (со всех продаж) |
| **Оценка дохода** | От $0 до $1M+/мес (Grammarly, Honey) |

**Особенности**:
- Самая большая аудитория
- Managed загрузка, платные расширения
- Поддержка subscriptions (с 2024)
- Строгая модерация (Manifest V3)

**Примеры успешных**: Grammarly (оценка $13B), Honey (куплен PayPal за $4B), LastPass

### Firefox Add-ons

## 2. CMS и веб-платформы

### WordPress Plugin Directory

| Параметр | Значение |
|---|---|
| **Аудитория** | 43% всех сайтов в интернете |
| **Язык** | PHP, JavaScript (React для блоков) |
| **Регистрация** | Бесплатно |
| **Доля** | 0% (бесплатный каталог) |
| **Монетизация** | Freemium: бесплатный плагин → продажа Pro-версии |

**Крупнейшие компании**: WooCommerce (Automattic), Yoast SEO ($20M+/год), Elementor ($100M+/год), Jetpack

**Платформы для продажи**:

| Платформа | Комиссия | Особенность |
|---|---|---|
| **WordPress.org** | 0% | Бесплатный каталог (freemium) |
| **WooCommerce Marketplace** | 4% + $75/год | Официальный |
| **Envato Market (CodeCanyon)** | до 55% → 12.5% (эксклюзив) | Крупнейший платный |
| **Freemius** | 9% + $0.50/сделка | Хостинг для WP-плагинов |
| **WooThemes** | 30% | Официальный |

**Факт**: CodeCanyon — крупнейший маркетплейс плагинов ($50M+/год выплат авторам)

**Стратегии монетизации**:
1. GitHub Sponsors (бесплатный плагин + спонсорство)
2. Open VSX (Eclipse Foundation)
3. Собственный веб-сайт с продажей Pro-версии
4. Open-core: базовый бесплатно, enterprise за деньги

## Чек-лист: запуск плагина

- [ ] Выбрать платформу с большой аудиторией
- [ ] Найти проблему, которую никто не решает (или решает плохо)
- [ ] Создать MVP (минимальный функционал)
- [ ] Добавить freemium-механику (база бесплатно, Pro за $)
- [ ] Продвигать: Product Hunt, Hacker News, Reddit, Twitter
- [ ] Собрать email-лист до запуска
- [ ] После запуска — активно работать с обратной связью
- [ ] Масштабироваться: добавить enterprise-фичи

> **Совет**: Самый прибыльный плагин — тот, который решает боль (pain point) для бизнеса. B2B плагины (Shopify, Salesforce) зарабатывают больше B2C (браузерные, Figma).
